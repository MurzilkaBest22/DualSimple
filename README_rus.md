ОПИСАНИЕ
DualSimple – упрощённая реализация автоматического дифференцирования (AD) на основе дуальных чисел. Поддерживает вычисление производных первого порядка и градиентов для функций одного и нескольких вещественных переменных. Эта библиотека является базовой версией для задач, не требующих производных высших порядков и дробного дифференцирования.

ВОЗМОЖНОСТИ
Класс Dual (дуальное число) с арифметическими операциями.
Основные элементарные функции (степенные, показательные, логарифмические, тригонометрические, обратные тригонометрические, гиперболические).
Оператор дифференцирования D(f, x) для функции одного аргумента.
Оператор дифференцирования D(f, X, index) для функции нескольких аргументов (частная производная).
Функция grad(f, X) для вычисления градиента.
Машинная точность, линейная сложность.

УСТАНОВКА И СБОРКА
Библиотека состоит из dual_simple.h и dual_simple.cpp. Для использования:
Скопируйте папку DualSimple в ваш проект.
Добавьте dual_simple.cpp к компиляции.
Включите заголовок:

#include "DualSimple/dual_simple.h"

Используйте пространство имён dual_simple.
Если вы используете статическую библиотеку, скомпилируйте dual_simple.cpp в DualSimple.lib и линкуйте её.

ЗАВИСИМОСТИ
RealUtils (для удобных имён функций).

ПРИМЕР

#include <iostream>
#include "DualSimple/dual_simple.h"
#include "RealUtils/real_utils.h"

auto f = [](dual_simple::Dual x) {
    return sin(x) * exp(x);
};

int main() {
    double x = 1.0;
    double df = dual_simple::D(f, x);
    std::cout << "f'(1) = " << df << std::endl;
    return 0;
}
